# libpathutils

Cross-platform filesystem path utility functions: 
- path sanitize
- path normalize
- create unique filename/path
- create all directories in path (`mkdir -p`)
- `char` UTF8 to UC2 `WCHAR` path string conversion and vice versa
- ...
